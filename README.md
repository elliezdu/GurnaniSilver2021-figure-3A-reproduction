# GurnaniSilver2021-figure-3A-reproduction

This project reproduces Figure 3A from the paper: Gurnani & Silver (2021), https://pubmed.ncbi.nlm.nih.gov/33848473/

## Overview
This notebook explores neural activity data and recreates a figure similar to one presented in the paper, including:
- PCA neural traces
- Behavioural state
- Speed & whisker motion index (wmi)
- Event markers


- Recreated using Python (NumPy, Matplotlib, Pandas)
- Focus on replicating data trends and visualization
